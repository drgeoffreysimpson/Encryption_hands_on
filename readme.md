# Encryption Homework
# CYB 2013

1) Generate your own keys and save them to files (call the generate_new_keys_and_write_to_file function)
2) Encypt a message with your PUBLIC key and save it to a binary file
3) Add YOUR PRIVATE KEY and your ENCRYPTED message to the github project
4) COMMIT and PUSH to GitHub
5) Bonus Points:  Decrypt the message in the **binary** file "geoffreys_secret.bin" using the gts keyset (provided in the project directory) (it was encrypted with the public key, so you need to decrypt it with the private key).  Put the message in the Harvey submit box when you submit your assignment.
